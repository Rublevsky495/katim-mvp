Катим MVP - последняя версия

Открыть:
1. Дважды кликнуть index.html
2. Или запустить локальный сервер из этой папки:
   python3 -m http.server 4174 --bind 127.0.0.1

Состав:
- index.html — разметка экранов
- css/base.css — общие стили, layout, nav, кнопки
- css/customer.css — клиентские экраны и flow
- css/executor.css — экраны исполнителя
- js/app.js — логика приложения
- assets/
